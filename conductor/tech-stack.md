# Tech Stack

## Current Stack (Mobile)
- **Programming Language:** Kotlin
- **UI Framework:** Jetpack Compose with Material Design 3
- **Graphics:** OpenGL-backed blurs and custom Canvas animations
- **PDF Generation:** pdfbox-android
- **Image Loading:** Coil (with SVG support via AndroidSVG)
- **Persistence:** Jetpack DataStore (Preferences)
- **Architecture:** MVVM (ViewModel) with Repository pattern
- **CI/CD Environment:** Google Cloud Build (gcloud)
